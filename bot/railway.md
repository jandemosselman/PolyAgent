# Polymarket Resolution Checker Bot

This bot monitors Polymarket traders and sends Telegram notifications.

## Environment Variables Required

- `TELEGRAM_BOT_TOKEN` - Your Telegram bot token from @BotFather
- `TELEGRAM_CHAT_ID` - Your Telegram chat ID
- `MONITORED_TRADERS` - Comma-separated trader addresses (e.g., 0x1234...,0x5678...)

## Deployment

This bot is designed to run 24/7 on Railway.app
